# HP Store Web Application

A web-based store management system built using Java and JSP.

## Tech Stack
- Java (JSP, JDBC)
- MySQL
- HTML, CSS, Bootstrap
- JavaScript, AJAX
- Apache Tomcat
- NetBeans IDE

## Features
- Admin panel
- Stock management
- Sales tracking
- User purchase flow

## Setup Instructions
1. Clone the repository
2. Import project into NetBeans
3. Configure Apache Tomcat
4. Create MySQL database manually
